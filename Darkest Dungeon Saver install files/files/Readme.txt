Note de l'auteur du logiciel (Haltarys)

DD Saver a été conçu pour sauvegarder des parties sur Darkest Dungeon, le jeu en ayant été volontairement privé par les devs.
Je ne garantis pas sa stabilité et ne suis pas responsable de l'usage que vous en faites (à chacun ses conneries !)
Je défends quiconque de s'en servir pour en tirer un bénéfice économique (ou alors versez-moi 99% des profits).
Ce programme est gratuit et n'a pas été conçu à des fins malveillantes : je ne collecte pas de données personnelles lorsque vous l'utilisez (mais méfiez-vous de Google, parce que lui le fait.)

Avant de vous servir de DD Saver, indiquez dans le fichier "DD Saves location.txt" dans quel dossier de Steam se situent les sauvegardes.
Emplacement par défaut : "C:\Program Files (x86)\Steam\userdata\[Steam User ID #]\262060"

Avant de vous servir de DD Saver, indiquez dans le fichier "DD url.txt" le chemin d'accès à l'application Darkest Dungeon.
Emplacement par défaut : "C:\Program Files (x86)\Steam\steamapps\common\DarkestDungeon\_windows\Darkest.exe"
Ou ""C:\Program Files (x86)\Steam\steamapps\common\DarkestDungeon\_windowsnosteam\Darkest.exe"

NB : il est possible qu'à cause d'une permission manquante, DD Saver n'ait pas l'autorisation pour lancer le jeu. Cependant, vous pouvez le lancer à partir d'un raccourci.
Exemple : "C:\Users\USERNAME\Desktop\Darkest Dungeon.lnk" ou ""C:\Users\USERNAME\Desktop\Darkest Dungeon.url"

Pour désinstaller DD Saver, supprimez simplement le dossier dans lequel le programme est installé.
Emplacement par défaut : "C:\Program Files (x86)\Darkest Dungeon Saver"
